    '''A custom vector plotting function that makes it easy to plot 2D vectors'''

            ymin = vectors[i][1]
        elif vectors[i][1] > ymax:
            ymax = vectors[i][1]

    buffer = 2
    plt.xlim(xmin - buffer, xmax + buffer)
    plt.ylim(ymin - buffer, ymax + buffer)

    plt.grid()

    if title != None:
        plt.title(title)

    plt.legend()

Introduction¶

When using matrices to represent data there are often specific things going on in the data that we care about. We care about very similar things, very different things and we also often like to be able to determine the specific features that best distinguish/classify the data.

Frequently to examine these differences or similarities more closely we will factor our data matrix such that the factored version has certain properties.

Principle Component Analysis (PCA)¶

PCA finds the principle components of a matrix. More technically, it is an algorithm that factors , we factor a matrix into the sum of a low rank matrix and sparse matrix.

There are many ways to think about PCA:

  • extracts information from a high-dimensional space by projecting it into a lower-dimensional sub-space
  • finds the best rank-$k$ estimate $L$ of $M$ (minimizing $\| M - L \|$ where $L$ has rank-$k$)

Note: There are many versions of PCA...

https://jakevdp.github.io/PythonDataScienceHandbook/05.09-principal-component-analysis.html

Now, we can inspect the principle components (vectors that show the principle axes) and the variance of each component. These component vectors represent the principle components of the data and the length of each of the components is an indication of how important each vector is in describing the data distribution.

We can see that if we plot pca.components_ alone, we get unit vectors that show the directions of data distribution.
